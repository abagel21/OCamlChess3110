val heuristic : Board.t -> string * int

heuristic pos returns a pair with the move that maximizes the point values obtained by the current player on a board pos, and the value is what the move is worth.

val max_turn : Board.t -> int -> string * int

max_turn pos depth determines the best value for the maximizing player on a board pos until the search has reached depth.

val min_turn : Board.t -> int -> string * int

min_turn pos depth determines the best value for the minimizing player on a board pos until the search has reached depth.

val minimax : Board.t -> int -> string

minimax pos depth takes in a board state pos and determines the best move by running through every legal move up to a certain depth while maximizing the player's valuation and minimizing the opponents.
